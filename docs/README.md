### 介绍
- 经典坦克大战简单还原，游戏模式为无尽版，支持单人游戏和双人游戏
- Unity 版本 2019.3.0a8
- C# 脚本在 Assets/Scripts/
- 素材资源在 Assets/GameResource/

### 操作
- 主界面：
    - ws或↑↓选择游戏模式
    - 空格进入游戏
- 游戏界面：
    - 单人游戏：
        - wsad或↑↓←→移动
        - 空格或回车开炮
    - 双人游戏：
        - 玩家1：
            - wsad移动
            - 空格开炮
        - 玩家2：
            - ↑↓←→移动
            - 回车开炮
